# Vin.ID — Beta Invitation Message

Short messages the founder can send to 5–10 testers. Replace `[LINK]` with the
production URL. Keep the docs links optional — the in-app flow is enough to
start.

## English

**Subject: Try Vin.ID (beta) — ~20–30 min, feedback welcome**

Hi [Name],

I'm building **Vin.ID** — an app to keep a vehicle's full history (maintenance,
issues, documents, reminders) in one place, and to create a **Vehicle Passport**
you can share with a buyer when selling or transferring a car.

It's an early **beta**, so expect rough edges. Could you try it for about
**20–30 minutes** and tell me what you think?

**Start here:** [LINK]

**What to try:**
- Create an account, add a vehicle, add some maintenance / an issue / a document.
- Create a **Vehicle Passport**, open its share link in a private window, and try
  "Save as PDF".
- If you can, use a **second account** (or a friend) to **accept** the Passport.
- Try it on your **phone**, and switch between **English and Hebrew**.

**A few honest notes:**
- The AI diagnosis and document "extraction" are **demo/mock** for now — not real
  AI/OCR.
- A Vehicle Passport is **not** an official ownership document or a mechanical
  certification. Please **don't use it for safety-critical decisions** — always
  inspect a car and consult a professional.

**Feedback:** in the app, go to **Settings → Send beta feedback**. Tell me what
was confusing, whether you'd trust/share a Passport, and how mobile felt.

Thank you so much 🙏
[Your name]
